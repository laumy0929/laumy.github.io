        端侧vscode AI开发环境搭建

        2025-06-18 · risc-v

        PC上安装vscodium

        下载安装vscodium，https://vscodium.com/

        安装Open Remote - SSH插件。

        可能要等等比较久，跟网络有关系，如果还是安装不了，可以参考一下这个办法使用离线的方式，https://zhuanlan.zhihu.com/p/671718415

        连上之后，安装插件

        设备端安装

        主要是在设备端安装python、jupyter等工具。

        sudo apt install -y python3-dev python3-venv
python3 -m venv ~/demo_env
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
pip config set global.extra-index-url https://git.spacemit.com/api/v4/projects/33/packages/pypi/simple
source ~/demo_env/bin/activate
pip list
deactivate

访问：
https://git.spacemit.com/archive/pypi/-/packages/
查看已经预编译的python包列表

pip install numpy==1.26.4 matplotlib==3.10.0 spacemit-ort opencv-python==4.11.0.86 scipy==1.14.1 scikit-learn==1.5.2

pip install –r notebook_requirements.txt

        运行

        运行代码，先选择运行环境，然后再运行。
